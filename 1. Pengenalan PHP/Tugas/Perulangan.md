# Tugas 1: Bilangan Pangkat

Buatlah deretan bilangan pangkat 2 dari bilangan 1-10 menggunakan perulangan (loop)

> Hasil yang diharapkan
>
> 1, 4, 9, 16, 25, 36, 49, 64, 81, 100,

<details>
    <summary>Jawaban</summary>

```php
for ($i=1; $i <= 10; $i++) { 
    echo $i * $i . ", ";
}
```
